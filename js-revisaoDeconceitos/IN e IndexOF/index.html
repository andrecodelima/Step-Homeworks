declaração

INDEX OF - BUSCANDO ELEMENTOS NO ARRAY

Sintaxe

string.indexOf(elemento)

Como Buscar um Elemento em Arrays: Método indexOf

Todo objeto do tipo Array possui um método próprio, nativo, o indexOf.
Para usá-lo, basta colocar um ponto após o nome do array, escrever indexOf
e passar como argumento o elemento que você deseja buscar.
Por exemplo, se quisermos encontrar o número 10 em um array chamado vetor:
vetor.indexOf(10);
